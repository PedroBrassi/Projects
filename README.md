# Computer Science Portfolio — Pedro Brassi Luccas

This repository gathers projects and assignments developed throughout my
Computer Science studies at **Universidade Federal de Alfenas (UNIFAL-MG)**,
as well as external courses taken outside of university. It spans algorithm
design and analysis, object-oriented programming, logic programming, and
image processing, in languages such as **C, Java and Prolog**.

Each folder contains its own `README.md` with a detailed description of the
problem, the approach taken, and instructions to build and run the code.

## Context

Most of these projects were developed for the **AEDS III** (Algorithms and
Data Structures III) course under Professor Iago A. Carvalho, covering
algorithmic complexity, graph algorithms, and heuristic/meta-heuristic
optimization. Others come from the **Object-Oriented Programming**, **Logic
Programming**, and **Image Processing** courses, and from external study
outside the university curriculum.
